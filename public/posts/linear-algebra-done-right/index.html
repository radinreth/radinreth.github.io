Complex Numbers#

It was invented so that we can take a square roots of negative numbers.

A square root of $-1$, denoted by $i$.

The set of all complex numbers is: $$ C = \{ a+bi : a, b \in \R \} $$

Addition and multiplication on $C$ $$ (a + bi) + (c + di) = (a + b) + (b + d)i \\ (a + bi)(c + di) = (ac - bd) + (ad + bc)i $$
